otosliderPlugin
===============

Otomatik Dönen Slider

$("#slider").otosliderPlugin({
			sec:2000,//donme suresi vere bilirsiniz});
		
Hem otomatik hemde butonlu slider ayarı

$("#slider").otosliderPlugin({
      sec:2000,//donme suresi vere bilirsiniz
      next:$("#next"),
      prev:$("#prev")
		});
